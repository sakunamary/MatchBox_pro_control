 HMI --> HB 的控制命令，主要从HMI模块传入的串口文本格式
 火力开关：HEAT,1 / 0\r
 冷却开关：COOL,1 / 0\r
 火力值：PWR,0-100\r
 pid开关：PID_MODE,1/0\r
 PID_SV(X10): PID_SV,2345\r
 PID_P(X100): PID_P,0\r
 PID_I(X100): PID_I,0\r
 PID_D(X100): PID_D,0\r




// HB --> HMI的数据帧 FrameLenght = 16
// 帧头: 69 FF
// 类型: 01温度数据
// 温度1: 00 00 // uint16
// 温度2: 00 00 // uint16
// 温度3: 00 00 // uint16
// 温度4: 00 00 // uint16
// NULL: 00 00
// 帧尾:FF FF FF

// 火力: 00 00 // uint16
// 火力开关: 00 00 // uint16
// 冷却开关: 00 00 // uint16
// NULL: 00 00 // uint16
// NULL: 00 00 // uint16
// 帧尾:FF FF FF

// 火力开关: 00 00// uint16
// 冷却开关: 00 00// uint16
// NULL: 00 00 // uint16
// NULL: 00 00 // uint16
// 帧尾:FF FF FF
